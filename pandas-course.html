1. Introduction to Pandas

1.1 Pandas kya hai

Pandas Python ki ek powerful library hai jo data analysis aur data manipulation ke liye use hoti hai.

Pandas ka naam aaya hai: Panel Data se (multi-dimensional data).

Pandas specially tab use hota hai jab:

  • Data table form me ho
  • Rows aur columns ke saath kaam ho
  • CSV, Excel, JSON jaise files handle karni ho

Simple words me:

Pandas = data ko samajhne, clean karne aur analyze karne ka tool.

1.2 Pandas kyun use hota hai

Real-world data hamesha:

  • Messy hota hai
  • Missing values hoti hain
  • Large size ka hota hai

Pandas use hota hai kyunki:

  • Large data easily handle karta hai
  • Fast data filtering aur sorting karta hai
  • Missing data handle karta hai
  • Excel / CSV files se direct kaam hota hai
  • Time-series data ke liye best hai

Example use-cases:

  • Student marks analysis
  • Sales report analysis
  • Bank transaction analysis
  • Stock market data analysis

1.3 Data Analysis me Pandas ka Role

Data Analysis ke major steps:

  1. Data Load karna
    CSV, Excel, Database se
  2. Data Cleaning
    • Missing values
    • Duplicate rows
    • Wrong data
  3. Data Transformation
    • Columns add/remove
    • Data type change
    • Grouping
  4. Data Analysis
    • Mean
    • Sum
    • Count
    • Trends & patterns
  5. Data Export
    CSV / Excel file

Ye sab kaam Pandas single library ke andar possible hai.

Exam Line:

Pandas is used for data cleaning, manipulation, and analysis.

1.4 Pandas vs NumPy vs Excel

Pandas

  • Tabular data (rows & columns)
  • Missing data handle karta hai
  • Label-based indexing
  • CSV, Excel file support
  • Data analysis ke liye best

NumPy

  • Numerical computation ke liye use hota hai
  • Fast mathematical operations
  • Fixed-type arrays
  • Scientific computing ke liye best

Excel

  • Small data ke liye easy
  • GUI based tool
  • Large data me slow hota hai
  • Automation difficult hoti hai

One-line Comparison

  • NumPy → numbers
  • Pandas → tables
  • Excel → manual analysis

2. Core Data Structures (Pandas)

2.1 Series kya hoti hai

Series Pandas ka ek 1-dimensional data structure hota hai.

Ye bilkul ek column jaisa hota hai jisme:

  • Values hoti hain
  • Har value ke saath ek index hota hai

Series ek labeled array hota hai.

Example:

import pandas as pd

s = pd.Series([10, 20, 30, 40])

Conceptual Output:

Index → 0   1   2   3
Value → 10  20  30  40

Default index 0 se start hota hai.

2.2 DataFrame kya hota hai

DataFrame Pandas ka ek 2-dimensional data structure hota hai.

Ye bilkul Excel sheet / table jaisa hota hai:

  • Rows
  • Columns
  • Different data types

Important baat: Har column ek Series hoti hai.

Example:

import pandas as pd

df = pd.DataFrame({
    "Name": ["Amit", "Neha", "Rahul"],
    "Age": [20, 21, 22],
    "Marks": [85, 90, 88]
})

Structure:

  • Rows → Students
  • Columns → Name, Age, Marks

ONE-LINE REVISION (Exam Ready)

  • Series → 1D labeled data
  • DataFrame → 2D tabular data
  • Series = single column
  • DataFrame = multiple Series

INTERVIEW ONE-LINER

“Series is a one-dimensional labeled array, while DataFrame is a two-dimensional tabular data structure made up of multiple Series.”

2.3 Series vs DataFrame

Series

  • 1-D data structure
  • Single column
  • Values + Index
  • List jaisa lagta hai

DataFrame

  • 2-D data structure
  • Multiple columns
  • Rows + Columns
  • Excel table jaisa hota hai

One-Line Difference

  • Series → single column
  • DataFrame → multiple columns

2.4 Index ka Concept

Index Pandas ka ek bahut hi important concept hai.

Index ka main kaam hota hai:

  • Data ko identify karna
  • Data ko fast access karna

Default Index

Agar aap index manually provide nahi karte, to Pandas automatically index assign karta hai: 

0, 1, 2, 3, ...

Custom Index

s = pd.Series([100, 200, 300], index=["a", "b", "c"])

Index Features

  • Index immutable hota hai (directly change nahi hota)
  • Label-based access allow karta hai
  • Data alignment me help karta hai (merge, join)

Index Access Example

s["b"]   # Output: 200

Why Index is Important

  • Fast data access
  • Data alignment (merge, join operations)
  • Meaningful labels ke through clarity

ONE-LINE REVISION (Exam Ready)

  • Series → 1D, single column
  • DataFrame → 2D, table
  • Index → data ka label
  • Default index → 0,1,2
  • Custom index → meaningful names

INTERVIEW ONE-LINER

“A Series is a one-dimensional labeled array, while a DataFrame is a two-dimensional tabular data structure made up of multiple Series, with index providing efficient data access.”

3. Creating Data (Pandas)

3.1 List se DataFrame

Jab data simple list ke form me hota hai, tab Pandas se DataFrame banaya ja sakta hai.

Single List → One Column DataFrame

import pandas as pd

data = [10, 20, 30, 40]

df = pd.DataFrame(data, columns=["Marks"])
print(df)

Output:

Marks
10
20
30
40

List of Lists → Multiple Columns

data = [
    ["Amit", 20, 85],
    ["Neha", 21, 90],
    ["Rahul", 22, 88]
]

df = pd.DataFrame(data, columns=["Name", "Age", "Marks"])
print(df)
  • List ke andar list → rows represent karti hain
  • columns parameter se column names dete hain

3.2 Dictionary se DataFrame

Dictionary Pandas ke liye best structure hota hai kyunki dictionary ka:

  • Key → column name
  • Value → column data

Example

data = {
    "Name": ["Amit", "Neha", "Rahul"],
    "Age": [20, 21, 22],
    "Marks": [85, 90, 88]
}

df = pd.DataFrame(data)
print(df)

Important Point

  • Dictionary keys = column names
  • Dictionary values = column data

3.3 NumPy Array se DataFrame

NumPy array se DataFrame banana tab useful hota hai jab data pure numerical ho.

Example

import numpy as np
import pandas as pd

arr = np.array([
    [10, 20, 30],
    [40, 50, 60],
    [70, 80, 90]
])

df = pd.DataFrame(arr, columns=["Math", "Science", "English"])
print(df)

Why NumPy + Pandas?

  • NumPy → fast numerical calculations
  • Pandas → tabular (row–column) structure

3.4 CSV, Excel, JSON File Read Karna

Real-world data mostly files ke form me hota hai:

  • CSV
  • Excel
  • JSON

Pandas in sab files ko easily DataFrame me read kar sakta hai.

read_csv()

CSV file read karne ke liye use hota hai.

pd.read_csv("file_name.csv")

Example

df = pd.read_csv("students.csv")
print(df)

Common Parameters

pd.read_csv("data.csv", header=None)
pd.read_csv("data.csv", skiprows=1)
pd.read_csv("data.csv", delimiter=",")

Exam Line: read_csv() CSV file ko DataFrame me convert karta hai.

read_excel()

Excel file (.xlsx) read karne ke liye use hota hai.

pd.read_excel("file_name.xlsx")

Example

df = pd.read_excel("students.xlsx")
print(df)

Sheet Selection

pd.read_excel("students.xlsx", sheet_name="Sheet1")

Note

Excel file read karne ke liye openpyxl library required hoti hai. 

pip install openpyxl

read_json()

JSON file read karne ke liye use hota hai.

df = pd.read_json("data.json")

ONE-LINE REVISION (Exam Ready)

  • List → DataFrame using pd.DataFrame()
  • Dictionary → Best structure for DataFrame
  • NumPy → numerical data
  • CSV → read_csv()
  • Excel → read_excel()
  • JSON → read_json()

Method 1: Bracket notation

df["Marks"]

Note

  • Dot notation tab use karo jab:
  • Column name me space na ho

Exam Line: Single column selection returns a Series.

6.2 Multiple Columns Selection

Multiple columns select karne par DataFrame milta hai.

Syntax

df[["Name", "Marks"]]

Important

  • Double square brackets zaroori hain

Exam Line: Multiple column selection returns a DataFrame.

6.3 Row Selection (Slicing)

Row slicing Python list slicing jaisa hota hai.

Syntax

df[start : end]

Example

df[0:5]     # first 5 rows
df[2:6]     # rows from index 2 to 5

Important Rule

  • End index include nahi hota

Use

  • Quick row filtering
  • Small data preview

6.4 Boolean Indexing

Condition laga kar rows filter karna Boolean Indexing kehlata hai.

Single Condition

df[df["Marks"] > 80]

Multiple Conditions (AND)

df[(df["Marks"] > 80) & (df["Age"] > 20)]

OR Condition

df[(df["Marks"] > 80) | (df["Age"] > 20)]

NOT Condition

df[~(df["Marks"] > 80)]

Important Rules

  • & (AND), | (OR), ~ (NOT) use karo
  • and / or use nahi karna
  • Har condition ke around brackets () lagao

Exam Line: Boolean indexing is used to filter data based on conditions.

Quick Summary (Very Important)

  • df["col"] → single column (Series)
  • df[["c1","c2"]] → multiple columns (DataFrame)
  • df[1:5] → row slicing
  • df[condition] → boolean filtering

Common Mistakes (Avoid Karo)

  • ❌ df["Name","Marks"]
  • ❌ df["Marks"] > 80 without df[...]
  • ❌ and / or instead of & / |
  • ❌ Missing brackets in conditions

ONE-LINE REVISION (Exam Ready)

  • Single column → Series
  • Multiple columns → DataFrame
  • Row slicing → start:end
  • Boolean indexing → condition based filter

INTERVIEW ONE-LINER

7. loc & iloc (Pandas)

Pandas me data selection ka sabse powerful aur safe tarika loc aur iloc hote hain.

Ye dono confusion clear kar dete hain:

  • row vs column
  • label vs position
  • slicing mistakes

7.1 loc (Label-based Access)

loc ka use hota hai labels (names) ke basis par data access karne ke liye.

Syntax

df.loc[row_label, column_label]

Examples

Single row

df.loc[0]

Single column

df.loc[:, "Marks"]

Specific row & column

df.loc[2, "Marks"]

Multiple rows & columns

df.loc[0:3, ["Name", "Marks"]]

Important Rule

  • loc me end label include hota hai

7.2 iloc (Index-based Access)

iloc ka use hota hai integer position (index number) ke basis par.

Syntax

df.iloc[row_index, column_index]

Examples

Single row

df.iloc[0]

Single column

df.iloc[:, 2]

Specific row & column

df.iloc[2, 1]

Multiple rows & columns

df.iloc[0:3, [0, 2]]

Important Rule

  • iloc me end index exclude hota hai
  • Ye pure Python slicing jaisa behave karta hai

7.3 loc vs iloc (Direct Comparison)

loc

  • Label-based
  • Row/column names use karta hai
  • End index include hota hai
  • Real-world data me zyada readable

iloc

  • Position-based
  • Integer index use karta hai
  • End index exclude hota hai
  • Fast & precise access

One-line:

  • loc = labels
  • iloc = positions

7.4 Boolean Indexing with loc

loc ke saath condition lagana best practice mani jaati hai.

Example

df.loc[df["Marks"] > 80, ["Name", "Marks"]]

Reason

  • Clean code
  • Warnings kam aati hain
  • Safe assignment hota hai

7.5 Common Mistakes & Fixes (Very Important)

Mistake 1: iloc me label use karna

❌ df.iloc["Marks"]

Fix:

df.iloc[:, 2]

Mistake 2: loc me integer ko position samajhna

❌ df.loc[1]   (jab index labels different ho)

Fix:

  • df.index check karo
  • Ya iloc use karo

Mistake 3: End index confusion

❌ df.iloc[0:3] ko include samajhna

Yaad rakho:

  • iloc → end exclude
  • loc → end include

Mistake 4: Chained indexing

❌ df[df["Marks"] > 80]["Grade"] = "A"

Fix (use loc):

df.loc[df["Marks"] > 80, "Grade"] = "A"

Best Practices

  • Labels ke liye loc use karo
  • Positions ke liye iloc use karo
  • Condition ke saath hamesha loc
  • Chained indexing avoid karo

ONE-LINE REVISION (Exam Ready)

  • loc → label based
  • iloc → index based
  • loc → end include
  • iloc → end exclude
  • Boolean + loc → best practice

INTERVIEW ONE-LINER

“loc is used for label-based data selection while iloc is used for integer position-based selection. Using loc with boolean conditions is the safest and most readable way to filter and update data in Pandas.”

8. Index Handling (Pandas)

Pandas me Index sirf numbering nahi hota, balki data ko identify aur manage karne ka powerful tool hota hai.

8.1 set_index()

Kisi column ko index banana ho, tab set_index() use hota hai.

Syntax

df.set_index("column_name")

Example

df = df.set_index("Name")
print(df)

Important Notes

  • Original index replace ho jata hai
  • By default original column remove ho jata hai

Keep column also

df.set_index("Name", drop=False)

Inplace change

df.set_index("Name", inplace=True)

Exam Line: set_index() sets an existing column as index.

8.2 reset_index()

Index ko wapas normal column banana ho tab reset_index() use hota hai.

Syntax

df.reset_index()

Example

df = df.reset_index()

Remove index column

df.reset_index(drop=True)

Inplace change

df.reset_index(inplace=True)

Exam Line: reset_index() resets index to default integer index.

8.3 Rename Index

Index ka naam ya values change karne ke liye rename-related methods use hote hain.

Rename Index Values

df.rename(index={0: "a", 1: "b"})

Rename Index Name (Label)

df.index.name = "Student_ID"

Rename Column Index (Axis)

df.rename_axis("Roll_No", axis="index")

8.4 MultiIndex Basics

MultiIndex ka matlab hota hai ek se zyada level ka index.

Isse hierarchical data represent hota hai.

Example

arrays = [
    ["Class1", "Class1", "Class2", "Class2"],
    ["A", "B", "A", "B"]
]

multi_index = pd.MultiIndex.from_arrays(
    arrays, names=("Class", "Section")
)

df = pd.DataFrame(
    {"Marks": [80, 85, 90, 95]},
    index=multi_index
)

Access MultiIndex Data

df.loc["Class1"]
df.loc[("Class2", "A")]

Why MultiIndex?

  • Complex hierarchical data handle karne ke liye
  • Group-wise analysis ke liye
  • Advanced data analytics ke liye

Common Mistakes (Avoid Karo)

  • Index ko ignore karna
  • set_index() ke baad reset_index() bhool jana
  • Label-based data me iloc use karna
  • MultiIndex ko complex samajhkar avoid karna

ONE-LINE REVISION (Exam Ready)

  • set_index → column to index
  • reset_index → index to column
  • rename → index rename
  • MultiIndex → multi-level index

INTERVIEW ONE-LINER

“Index handling in Pandas involves setting, resetting, and renaming indexes, and using MultiIndex to represent hierarchical data efficiently for advanced analysis.”

9. Sorting & Filtering Data (Pandas)

Real-world data analysis me data ko sort aur filter karna bahut common aur important task hota hai.

Sorting ka matlab: order me arrange karna Filtering ka matlab: condition ke basis par data nikalna

9.1 sort_values()

sort_values() ka use column ke values ke basis par data ko sort karne ke liye hota hai.

Syntax

df.sort_values(by="column_name")

Example

df.sort_values(by="Marks")

Descending Order

df.sort_values(by="Marks", ascending=False)

Multiple Columns Sorting

df.sort_values(by=["Age", "Marks"], ascending=[True, False])

Inplace Sorting

df.sort_values(by="Marks", inplace=True)

Exam Line: sort_values() sorts data by column values.

9.2 sort_index()

sort_index() ka use index ke basis par data ko sort karne ke liye hota hai.

Syntax

df.sort_index()

Descending Index Order

df.sort_index(ascending=False)

Use case: Jab index meaningful ho (dates, IDs)

Exam Line: sort_index() sorts data by index.

9.3 Filtering using Conditions

Condition laga kar data select karna filtering kehlata hai.

Single Condition

df[df["Marks"] > 80]

Multiple Conditions (AND)

df[(df["Marks"] > 80) & (df["Age"] > 20)]

OR Condition

df[(df["Marks"] > 80) | (df["Age"] > 20)]

NOT Condition

df[~(df["Marks"] > 80)]

Rule: & | ~ use karo, and / or nahi

9.4 query() Method ⭐

query() ek clean aur readable filtering method hai.

Syntax

df.query("Marks > 80")

Multiple Conditions

df.query("Marks > 80 and Age > 20")

Using Variables

min_marks = 75
df.query("Marks > @min_marks")

Why query()?

  • Readable syntax
  • No bracket confusion
  • SQL-like feel

Note: Column names me space ho to backticks (` `) use karo

9.5 isin()

isin() ka use multiple values match karne ke liye hota hai.

Example

df[df["Grade"].isin(["A", "B"])]

Use case: Category-based filtering

Common Mistakes (Avoid Karo)

  • sort_values() me column name bhoolna
  • and / or use karna instead of & |
  • query() me variable ke saath @ na lagana
  • sort_index aur sort_values confuse karna

Quick Comparison

  • sort_values → column based sorting
  • sort_index → index based sorting
  • filtering → condition based selection
  • query → clean filtering syntax

ONE-LINE REVISION (Exam Ready)

  • sort_values → column order
  • sort_index → index order
  • filter → condition based
  • query → readable filter

INTERVIEW ONE-LINER

“In Pandas, sorting is done using sort_values and sort_index, while filtering can be achieved using boolean conditions or the query method for clean and readable data selection.”

10. Handling Missing Data (Pandas)

Real-world data me missing values bahut common hoti hain. Isliye data analysis se pehle unko handle karna mandatory hota hai.

Missing data ka matlab:

  • Value available nahi hai
  • Data incomplete hai
  • NaN / None / NA present hai

10.1 Missing Values Types

  • NaN → Not a Number (most common)
  • None → Python null value
  • NaT → Not a Time (datetime)

Important: Pandas internally sabko NaN ki tarah treat karta hai.

10.2 Checking Missing Values

isna() / isnull()

Check karta hai ki value missing hai ya nahi. 

df.isna()
df.isnull()

Example

df.isna().sum()

=> Har column me kitni missing values hain

Exam Line: isna() detects missing values.

10.3 notna() / notnull()

Non-missing values check karta hai. 

df.notna()
df.notnull()

10.4 Dropping Missing Data

dropna()

Missing values wale rows/columns remove karta hai.

Drop rows (default)

df.dropna()

Drop columns

df.dropna(axis=1)

Drop if all values are NaN

df.dropna(how="all")

Drop if any value is NaN

df.dropna(how="any")

Inplace drop

df.dropna(inplace=True)

Warning: Data loss ho sakta hai

10.5 Filling Missing Data

fillna()

Missing values ko kisi value se replace karta hai.

Fill with constant value

df.fillna(0)

Column-wise filling

df["Marks"].fillna(df["Marks"].mean())

Multiple columns fill

df.fillna({
  "Marks": df["Marks"].mean(),
  "Age": df["Age"].median()
})

Forward Fill (ffill)

df.fillna(method="ffill")

Backward Fill (bfill)

df.fillna(method="bfill")

Exam Line: fillna() replaces missing values.

10.6 Interpolation

Numeric data ke liye smooth filling karta hai. 

df.interpolate()

Use case: time-series data

10.7 Replace vs Fillna

replace()

Specific values ko replace karta hai. 

df.replace(-1, np.nan)

replace → specific value fillna → only NaN

Best Practices (Interview Favorite)

  • Pehele missing values count karo
  • Drop tabhi karo jab data useless ho
  • Numerical → mean / median
  • Categorical → mode
  • Time-series → ffill / interpolate

Quick Comparison

  • isna → missing detect
  • dropna → remove missing
  • fillna → replace missing
  • interpolate → smooth fill

  • NaN → missing value
  • isna → detect
  • dropna → remove
  • fillna → replace
  • interpolate → numeric fill
