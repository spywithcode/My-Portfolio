1.1 C kya hai aur kyun important

C kya hai?

C ek programming language hai jiska use computer ko instructions dene ke liye hota hai. Isse hum software, system programs aur hardware-related programs bana sakte hain.

Simple words me:

üëâ C = Computer ko direct aur fast tareeke se kaam karwane wali language

C ek Procedural Language hai

Procedural language ka matlab hota hai:

  • Program ko steps (procedures / functions) me likhna
  • Execution line by line hota hai

Real-life example (Chai banana)

  • Gas on karo
  • Pani daalo
  • Chai patti daalo
  • Doodh daalo

Isi tarah C program bhi step-by-step chalta hai.

C example (Procedural Style)

#include <stdio.h>

int main() {
    int a = 5, b = 3;
    int sum = a + b;
    printf("Sum = %d", sum);
    return 0;
}

Yahan kya ho raha hai?

  • Pehle variables bane
  • Phir calculation hui
  • Phir output print hua

C fast kyun hoti hai?

C fast hoti hai kyunki:

  • Ye hardware ke bahut close hoti hai
  • Extra layers (overhead) kam hoti hain
  • Memory par direct control milta hai

Simple comparison

  • Python / Java ‚Üí slow but easy
  • C ‚Üí thodi hard but bahut fast

Isi liye C ka use hota hai:

  • Games
  • Operating Systems
  • Embedded Systems

C hardware ke close kaise hai?

C me hum:

  • Memory address use kar sakte hain
  • Pointers ke through RAM control kar sakte hain

Example (Pointers)

int x = 10;
int *p = &x;

Matlab:

  • x memory me kahin store hai
  • p uska address store kar raha hai

Isi liye C ko low-level language jaisa feel aata hai.

C kyun important hai?

1. Operating System ka base

  • Linux
  • Windows (core parts)
  • UNIX

üëâ Ye sab C me likhe gaye hain.

2. Drivers & Embedded Systems

  • Printer drivers
  • Keyboard drivers
  • Microcontroller programs

üëâ Mostly C me hote hain.

3. Baaki languages ka base

Agar C samajh aa gaya, to:

  • C++
  • Java
  • Python
  • Go

üëâ Inhe samajhna easy ho jata hai.

Simple Real-life Example (C ka use)

Socho:

üëâ Tumhe ek machine ko bolna hai: ‚ÄúButton dabate hi LED on karo‚Äù

Ye kaam C + Embedded System se hota hai, Python se nahi.

Exam / Interview Ready Lines

  • C is a procedural programming language
  • C is fast and close to hardware
  • C is used in OS, drivers, and embedded systems
  • Learning C makes other languages easier

One-line Summary (Yaad rakhne ke liye)

C ek fast, procedural, hardware-close language hai jo system programming ka base hai.

1.2 First C Program

Is section me hum pehla C program samjhenge aur ye bhi dekhenge ki program kaise run hota hai.

First C Program

#include <stdio.h>

int main() {
    printf("Hello, World!");
    return 0;
}

Ye C ka sabse basic aur first program mana jata hai. Ab iske har part ko simple language me samjhte hain.

#include <stdio.h>

Ye ek header file hai.

stdio.h ka matlab:

  • Standard Input Output
  • Input lene aur output dikhane ke functions deta hai

üëâ printf() function isi header file me defined hota hai.

Simple analogy:

Jaise mobile charger lagaye bina phone charge nahi hota, waise hi stdio.h include kiye bina printf() ka use nahi kar sakte.

main() function

main() C program ka starting point hota hai.

  • Program execution yahin se start hota hai
  • Har C program me main() hona zaroori hai
int main() {
    // code
}

üëâ int ka matlab:

  • main() ek integer value return karega
  • Usually return 0; likhte hain

return 0 ka matlab:

üëâ Program successfully run ho gaya.

printf()

printf() ka use hota hai screen par output dikhane ke liye. 

printf("Hello, World!");

üëâ Jo bhi double quotes ke andar likha hota hai, wo screen par print hota hai.

Example:

Output:

Why semicolon (;) is important?

C me har statement ke end me semicolon ( ; ) lagta hai.

üëâ Ye compiler ko batata hai ki statement yahin khatam ho gayi.

Compilation Flow

C program directly run nahi hota, pehle usse compile karna padta hai.

Flow:

.c file  ‚Üí  Compiler  ‚Üí  .exe file
  • .c ‚Üí Source code (human readable)
  • Compiler ‚Üí Code ko machine language me convert karta hai
  • .exe ‚Üí Executable file (computer run karta hai)

Example:

hello.c  ‚Üí  compile  ‚Üí  hello.exe

üëâ Windows me .exe file run hoti hai üëâ Linux me executable file run hoti hai

Overall Program Flow (Easy Language)

  • Header file include hoti hai
  • main() function start hota hai
  • printf() output show karta hai
  • return 0 se program end hota hai

ONE-LINE REVISION (Exam Ready)

  • #include <stdio.h> ‚Üí input/output functions
  • main() ‚Üí program execution start point
  • printf() ‚Üí output print karta hai
  • .c ‚Üí source file
  • .exe ‚Üí executable file

INTERVIEW ONE-LINER

‚ÄúThe main function is the entry point of a C program, stdio.h provides input-output functions like printf, and a C program is compiled from a .c file into an executable file before execution.‚Äù

1.4 Variables & Constants (C Programming)

Variables aur Constants C programming ke basic building blocks hote hain.

Inka kaam hota hai:

  • Data ko memory me store karna
  • Program ke execution ke dauran use karna
  • Calculations aur logic apply karna

1.4.1 Variables kya hote hain?

Variable ek aisa container hota hai jo memory me data store karta hai aur jiska value program ke dauran change ho sakta hai.

Simple words me:

üëâ Variable = value rakhne ka box, jiska content change ho sakta hai

Real-life example:

Jaise ek bottle me pani bhara hai. Tum pani kam ya zyada kar sakte ho.

Variable Declaration

Variable declare karte waqt compiler ko batate hain:

  • Variable ka type
  • Variable ka naam

Syntax

data_type variable_name;

Example

int age;
float salary;
char grade;

üëâ Yahan sirf variable bana hai, value assign nahi hui.

Variable Initialization

Initialization ka matlab hota hai variable ko value dena.

Syntax

data_type variable_name = value;

Example

int age = 20;
float salary = 25000.50;
char grade = 'A';

üëâ Declaration + Initialization ek saath bhi ho sakta hai.

Declaration vs Initialization

  • Declaration ‚Üí variable banana
  • Initialization ‚Üí value dena

Complete Example

#include <stdio.h>

int main() {
    int a = 10;
    int b = 20;
    int sum = a + b;

    printf("Sum = %d", sum);
    return 0;
}

üëâ Yahan:

  • a, b, sum variables hain
  • Inki values program ke andar change ho sakti hain

1.4.2 Constants kya hote hain?

Constant wo value hoti hai jo program ke dauran change nahi hoti.

Simple words me:

üëâ Constant = fixed value

Real-life example:

Jaise:

  • PI = 3.14
  • Days in a week = 7

Ye values change nahi hoti.

const Keyword

C me constant banane ke liye const keyword use hota hai.

Syntax

const data_type variable_name = value;

Example

const int max_marks = 100;
const float pi = 3.14;

üëâ Ab inki value change karne ki koshish karoge to error aayega.

Invalid Example (Error)

const int x = 10;
x = 20;   // Error

üëâ const variable ki value change nahi ho sakti.

Why Use Constants?

  • Accidental change se protection
  • Program readable banta hai
  • Fixed values clear hoti hain
  • Debugging easy hoti hai

Variable vs Constant

  • Variable ‚Üí value change ho sakti hai
  • Constant ‚Üí value fixed hoti hai

ONE-LINE REVISION (Exam Ready)

  • Variable ‚Üí value can change
  • Declaration ‚Üí variable creation
  • Initialization ‚Üí assigning value
  • const ‚Üí fixed value
  • Constant ‚Üí value cannot change

INTERVIEW ONE-LINER

‚ÄúIn C, variables are used to store data whose values can change during program execution, while constants are declared using the const keyword and hold fixed values that cannot be modified.‚Äù

1.5 Operators (C Programming)

Operators C programming me wo symbols hote hain jo kisi value ya variable par operation perform karte hain.

Simple words me:

üëâ Operator = kaam karne wala sign

Example (real-life):

Jaise:

  • +
  • -
  • *
  • /

Ye sab calculation ke operators hain.

Types of Operators in C

  • Arithmetic Operators
  • Relational Operators
  • Logical Operators
  • Assignment Operators
  • Increment / Decrement Operators
  • Bitwise Operators (basic intro)

1.5.1 Arithmetic Operators

Arithmetic operators ka use math calculations ke liye hota hai.

Operators List

  • + ‚Üí Addition
  • - ‚Üí Subtraction
  • * ‚Üí Multiplication
  • / ‚Üí Division
  • % ‚Üí Modulus (remainder)

Example

int a = 10, b = 3;

printf("%d", a + b);  // 13
printf("%d", a - b);  // 7
printf("%d", a * b);  // 30
printf("%d", a / b);  // 3
printf("%d", a % b);  // 1

üëâ % sirf integer ke saath kaam karta hai.

1.5.2 Relational Operators

Relational operators ka use comparison ke liye hota hai.

Result hamesha:

  • 1 ‚Üí true
  • 0 ‚Üí false

Operators List

  • == ‚Üí equal to
  • != ‚Üí not equal to
  • > ‚Üí greater than
  • < ‚Üí less than
  • >= ‚Üí greater than or equal to
  • <= ‚Üí less than or equal to

Example

#include 

int main() {
    int a = 10, b = 5;

    printf("%d\n", (a > 5 && b > 3));    // 1
    printf("%d\n", (a > 20 || b > 3));   // 1
    printf("%d\n", !(a == 10));          // 0

    return 0;
}

1.5.3 Logical Operators

Logical operators ka use multiple conditions ko combine karne ke liye hota hai.

Operators List

  • && ‚Üí AND
  • || ‚Üí OR
  • ! ‚Üí NOT

Truth Meaning

  • AND ‚Üí dono true hone chahiye
  • OR ‚Üí koi ek true ho
  • NOT ‚Üí result ko ulta kar deta hai

Example

int a = 10, b = 5;

printf("%d", (a > 5 && b > 3));  // 1
printf("%d", (a > 20 || b > 3)); // 1
printf("%d", !(a == 10));        // 0

1.5.4 Assignment Operators

Assignment operators ka use value assign karne ke liye hota hai.

Operators List

  • = ‚Üí assign
  • += ‚Üí add and assign
  • -= ‚Üí subtract and assign
  • *= ‚Üí multiply and assign
  • /= ‚Üí divide and assign

Example

int a = 10;

a += 5;   // a = 15
a -= 2;   // a = 13
a *= 2;   // a = 26

1.5.5 Increment / Decrement Operators

Ye operators value ko 1 se increase ya decrease karte hain.

Operators

  • ++ ‚Üí increment
  • -- ‚Üí decrement

Types

  • Pre-increment (++a)
  • Post-increment (a++)

Example

int a = 5;

printf("%d", ++a); // 6
printf("%d", a++); // 6
printf("%d", a);   // 7

üëâ Pre pehle value badhata hai, post baad me.

1.5.6 Bitwise Operators (Basic Intro)

Bitwise operators ka use binary (0 aur 1) level par hota hai.

Mostly:

  • Low-level programming
  • Embedded systems
  • Performance optimization

Basic Operators

  • & ‚Üí Bitwise AND
  • | ‚Üí Bitwise OR
  • ^ ‚Üí Bitwise XOR
  • << ‚Üí Left shift
  • >> ‚Üí Right shift

Simple Example

int a = 5;   // 0101
int b = 3;   // 0011

printf("%d", a & b); // 1

üëâ Abhi sirf intro samjho, detail baad me.

ONE-LINE REVISION (Exam Ready)

  • Arithmetic ‚Üí calculation
  • Relational ‚Üí comparison
  • Logical ‚Üí conditions
  • Assignment ‚Üí value assign
  • ++ / -- ‚Üí increase / decrease
  • Bitwise ‚Üí binary operations

INTERVIEW ONE-LINER

‚ÄúOperators in C are symbols that perform operations on variables and values, including arithmetic, relational, logical, assignment, increment-decrement, and bitwise operations.‚Äù

2. Control Statements (Decision + Flow)

Control Statements ka kaam hota hai program ke flow ko control karna.

Simple words me:

üëâ Control statements decide karte hain:

  • Kaunsa code chalega
  • Kab chalega
  • Kis condition par chalega

Real-life example:

Agar barish ho rahi hai ‚Üí chhata le jao Warna ‚Üí chhata mat le jao

Isi tarah C program me bhi decision lena padta hai.

2.1 Conditional Statements

Conditional statements ka use decision making ke liye hota hai.

Matlab:

üëâ Agar condition true ho ‚Üí ek kaam üëâ Agar condition false ho ‚Üí dusra kaam

Types of Conditional Statements in C

  • if
  • if-else
  • else-if ladder
  • switch

2.1.1 if Statement

if statement tab use hota hai jab sirf ek condition check karni ho.

Syntax

if (condition) {
    // code
}

Example

int age = 20;

if (age >= 18) {
    printf("You are eligible to vote");
}

üëâ Agar condition true hui to code chalega üëâ Agar false hui to kuch nahi hoga

2.1.2 if-else Statement

if-else tab use hota hai jab dono cases handle karne ho.

Syntax

if (condition) {
    // true block
} else {
    // false block
}

Example

int marks = 35;

if (marks >= 40) {
    printf("Pass");
} else {
    printf("Fail");
}

üëâ Condition true ‚Üí if block üëâ Condition false ‚Üí else block

2.1.3 else-if Ladder

else-if ladder tab use hoti hai jab multiple conditions check karni ho.

Syntax

if (condition1) {
    // code
}
else if (condition2) {
    // code
}
else if (condition3) {
    // code
}
else {
    // default
}

Example

int marks = 85;

if (marks >= 90) {
    printf("Grade A");
}
else if (marks >= 75) {
    printf("Grade B");
}
else if (marks >= 50) {
    printf("Grade C");
}
else {
    printf("Fail");
}

üëâ Pehli true condition ka block execute hota hai üëâ Baaki conditions skip ho jaati hain

2.1.4 switch Statement

switch statement tab use hota hai jab ek hi variable ke liye multiple fixed options ho.

Mostly menu-driven programs me use hota hai.

Syntax

switch (expression) {
    case value1:
        // code
        break;
    case value2:
        // code
        break;
    default:
        // code
}

Example

int choice = 2;

switch (choice) {
    case 1:
        printf("Addition");
        break;
    case 2:
        printf("Subtraction");
        break;
    case 3:
        printf("Multiplication");
        break;
    default:
        printf("Invalid choice");
}

Important Points

  • switch me break zaroori hota hai
  • break na ho to fall-through hota hai
  • default optional hota hai but recommended

if-else vs switch (Basic Difference)

  • if-else ‚Üí range aur conditions ke liye
  • switch ‚Üí fixed values ke liye

Example:

marks > 40 ‚Üí if-else better menu choice (1,2,3) ‚Üí switch better

Common Mistakes (Avoid Karo)

  • ‚ùå if ke baad semicolon lagana
  • ‚ùå condition me = use karna (== ki jagah)
  • ‚ùå switch me break bhoolna
  • ‚ùå else-if ladder me order galat rakhna

ONE-LINE REVISION (Exam Ready)

  • if ‚Üí single condition
  • if-else ‚Üí true / false handling
  • else-if ‚Üí multiple conditions
  • switch ‚Üí fixed choices

INTERVIEW ONE-LINER

‚ÄúConditional statements in C such as if, if-else, else-if ladder, and switch are used to make decisions and control the flow of execution based on conditions.‚Äù

2.2 Looping Statements (C Language)

Looping ka matlab hota hai ek hi kaam ko baar-baar repeat karna jab tak koi condition true ho.

Simple words me:

üëâ Loop = repetition üëâ Jab tak condition true ‚Üí code repeat üëâ Condition false ‚Üí loop stop

Real-life example:

Class me attendance lena Har student ka naam bolna Jab tak students khatam na ho jaaye

Why Looping is Needed?

  • Code repeat likhne se bachata hai
  • Program short aur clean hota hai
  • Large data easily handle hota hai

Without loop: 

printf("Hello");
printf("Hello");
printf("Hello");
printf("Hello");

With loop: 

for(int i=1; i<=4; i++) {
    printf("Hello");
}

Types of Loops in C

  • for loop
  • while loop
  • do-while loop
  • Infinite loop

2.2.1 for Loop

for loop tab use hota hai jab number of iterations pehle se pata ho.

Syntax

for(initialization; condition; increment/decrement) {
    // code
}

Example

int i;

for(i = 1; i <= 5; i++) {
    printf("%d ", i);
}

Output: 

1 2 3 4 5

Working

  • Initialization ‚Üí ek baar
  • Condition ‚Üí har iteration se pehle
  • Increment/Decrement ‚Üí har iteration ke baad

2.2.2 while Loop

while loop tab use hota hai jab condition pehle check karni ho.

Syntax

while(condition) {
    // code
}

Example

int i = 1;

while(i <= 5) {
    printf("%d ", i);
    i++;
}

Output: 

1 2 3 4 5

Important Point

  • Pehle condition check hoti hai
  • Agar condition false hui ‚Üí loop ek baar bhi nahi chalega

2.2.3 do-while Loop

do-while loop tab use hota hai jab loop ko kam se kam ek baar chalana zaroori ho.

Syntax

do {
    // code
} while(condition);

Example

int i = 1;

do {
    printf("%d ", i);
    i++;
} while(i <= 5);

Output: 

1 2 3 4 5

Important Point

  • Pehle code execute hota hai
  • Condition baad me check hoti hai

for vs while vs do-while

  • for ‚Üí fixed iterations
  • while ‚Üí condition based
  • do-while ‚Üí at least once execution

2.2.4 Infinite Loop

Infinite loop wo loop hota hai jo kabhi terminate nahi hota.

Matlab:

üëâ Condition hamesha true üëâ Program continuously chalta rehta hai

Example (for loop)

for(;;) {
    printf("Hello");
}

Example (while loop)

while(1) {
    printf("Running");
}

Real-life Use

  • Operating systems
  • Servers
  • Embedded systems

Danger

  • Program hang ho sakta hai
  • CPU usage high ho jaata hai

Common Loop Mistakes (Avoid Karo)

  • ‚ùå Increment / decrement bhoolna
  • ‚ùå Infinite loop accidentally banana
  • ‚ùå Semicolon lagana: while(condition);

ONE-LINE REVISION (Exam Ready)

  • for ‚Üí known iterations
  • while ‚Üí condition first
  • do-while ‚Üí at least once
  • Infinite loop ‚Üí never ends

INTERVIEW ONE-LINER

‚ÄúLooping statements in C such as for, while, and do-while are used to repeatedly execute a block of code based on conditions, and infinite loops run continuously until externally stopped.‚Äù

2.3 Jump Statements (C Language)

Jump statements ka kaam hota hai program ke normal flow ko achanak change karna.

Simple words me:

üëâ Jump statement = control ko ek jagah se dusri jagah le jana üëâ Loop ya block ko todna ya skip karna

C me main jump statements hote hain:

  • break
  • continue
  • goto (avoid karna chahiye)

2.3.1 break Statement

break ka use hota hai loop ya switch ko turant terminate karne ke liye.

Matlab:

üëâ Condition mili üëâ break execute hua üëâ Loop wahi par band

Example (with loop)

int i;

for(i = 1; i <= 10; i++) {
    if(i == 5) {
        break;
    }
    printf("%d ", i);
}

Output: 

1 2 3 4

Explanation:

  • i = 5 hone par break chala
  • Loop turant stop ho gaya

Example (with switch)

int choice = 2;

switch(choice) {
    case 1:
        printf("One");
        break;
    case 2:
        printf("Two");
        break;
    default:
        printf("Invalid");
}

üëâ switch me break na lagao to fall-through ho jata hai.

2.3.2 continue Statement

continue ka use hota hai current iteration skip karne ke liye.

Matlab:

üëâ continue mila üëâ Neeche ka code skip üëâ Next iteration start

Example

int i;

for(i = 1; i <= 5; i++) {
    if(i == 3) {
        continue;
    }
    printf("%d ", i);
}

Output: 

1 2 4 5

Explanation:

  • i = 3 par continue chala
  • 3 print nahi hua
  • Loop aage badh gaya

Difference: break vs continue

  • break ‚Üí loop end
  • continue ‚Üí iteration skip

2.3.3 goto Statement

goto statement program control ko direct kisi label par le jaata hai.

Syntax

goto label;

label:
    // code

Example

#include 

int main() {
    int i = 1;

start:
    printf("%d ", i);
    i++;

    if(i <= 5)
        goto start;

    return 0;
}

Output: 

1 2 3 4 5

goto kyun avoid karna chahiye?

goto technically kaam karta hai, lekin use karna strongly avoid kiya jaata hai.

Reasons

  • Code confusing ho jata hai
  • Program flow samajhna mushkil
  • Debugging hard ho jaati hai
  • Spaghetti code ban jaata hai

Industry rule:

üëâ "Never use goto unless absolutely necessary"

Modern programming me:

  • Loops
  • Functions
  • break / continue

goto ka kaam safely kar dete hain.

Real-Life Analogy

  • break ‚Üí meeting turant khatam
  • continue ‚Üí is topic ko skip karo, next pe jao
  • goto ‚Üí bina order ke kisi bhi point pe chale jaana (confusing)

Common Mistakes (Avoid Karo)

  • ‚ùå break ko if ke bahar samajhna
  • ‚ùå continue se loop end samajhna
  • ‚ùå goto ka overuse

ONE-LINE REVISION (Exam Ready)

  • break ‚Üí loop / switch end
  • continue ‚Üí current iteration skip
  • goto ‚Üí direct jump (avoid)

INTERVIEW ONE-LINER

‚ÄúJump statements in C like break and continue are used to control loop execution, while goto allows unconditional jumps but is avoided due to poor readability and maintenance issues.‚Äù

üëâ Practice Tips (C Programming ‚Äì Beginner Level)

C programming me strong hone ka sabse best tareeka hai: basic logic wale programs khud likhna.

Neeche diye gaye questions har beginner ko mandatory practice karne chahiye.

1) Number Check Programs

Ye programs aapki if / else aur operators ki understanding clear karte hain.

Practice Questions

  • Even ya Odd number check karo
  • Positive, Negative ya Zero check karo
  • Number divisible by 5 ya 11 hai ya nahi

Example: Even / Odd

#include <stdio.h>

int main() {
    int n;
    printf("Enter number: ");
    scanf("%d", &n);

    if(n % 2 == 0)
        printf("Even number");
    else
        printf("Odd number");

    return 0;
}

üëâ Isme use hua:

  • if-else
  • modulus operator (%)

2) Table Print Program

Table print karna loops samajhne ke liye best exercise hai.

Practice Questions

  • Any number ka table print karo
  • Table reverse order me print karo

Example: Table of a Number

#include <stdio.h>

int main() {
    int i, n;
    printf("Enter number: ");
    scanf("%d", &n);

    for(i = 1; i <= 10; i++) {
        printf("%d x %d = %d\n", n, i, n*i);
    }

    return 0;
}

üëâ Isme use hua:

  • for loop
  • multiplication

3) Prime Number Program

Prime number program se aapka loop + condition logic strong hota hai.

Prime Number Definition

üëâ Jo number sirf 1 aur khud se divisible ho

Example: Prime Check

#include <stdio.h>

int main() {
    int n, i, flag = 0;

3.3 Call by Value vs Call by Reference (C Programming)

Function calling ke do tareeke hote hain: Call by Value aur Call by Reference.

Ye topic exam + interview + real coding ke liye bahut important hai.

Call by Value

üëâ Call by Value me:

  • Function ko variable ki copy pass hoti hai
  • Original variable change nahi hota

Simple Example

#include <stdio.h>

    x = 100;
}

    int a = 10;
    change(a);
    printf("a = %d", a);
    return 0;
}

Output: 

a = 10

üëâ Explanation:

  • a ki value = 10
  • x ko a ki copy mili
  • x change hua, a nahi

Call by Reference

üëâ Call by Reference me:

  • Function ko variable ka address pass hota hai
  • Original variable change ho jata hai

üëâ C me call by reference pointers se hota hai.

Example

#include <stdio.h>

void change(int *x) {
    *x = 100;
}

int main() {
    int a = 10;
    change(&a);
    printf("a = %d", a);
    return 0;
}

Output: 

a = 100

üëâ Explanation:

  • a ka address pass hua
  • *x se direct memory change hui
  • a ki value change ho gayi

Call by Value vs Call by Reference (Comparison)

Point Call by Value Call by Reference
Data passed Value copy Address
Original variable Not changed Changed
Pointers Not used Used
Safety Safe Risky if misused

Exam Ready Line

Call by value passes a copy of variable, while call by reference passes the address allowing modification of original data.

3.4 Recursion (C Programming)

üëâ Recursion ka matlab hota hai:

Two Important Parts of Recursion

üëâ Wo condition jahan recursion rukta hai.

‚ùå Base condition nahi hogi ‚Üí infinite recursion

2. Recursive Call

üëâ Function ka khud ko call karna.

Simple Recursion Example (Factorial)

Factorial:

5! = 5 √ó 4 √ó 3 √ó 2 √ó 1 

#include <stdio.h>

int fact(int n) {
    if (n == 1)      // base condition
        return 1;
    else
        return n * fact(n - 1);
}

int main() {
    int result;
    result = fact(5);
    printf("Factorial = %d", result);
    return 0;
}

Recursion me function calls stack me store hoti hain.

fact(5) call flow: 

fact(5)
 ‚Üí 5 * fact(4)
 ‚Üí 5 * 4 * fact(3)
 ‚Üí 5 * 4 * 3 * fact(2)
 ‚Üí 5 * 4 * 3 * 2 * fact(1)
 ‚Üí return 1

üëâ Ab stack reverse order me empty hota hai.

  • ‚ùå Zyada recursion ‚Üí stack overflow

Recursion vs Loop

Recursion Loop
Function calls itself Repeats using loop
Uses stack memory No stack usage
Code short Code faster

ONE-LINE REVISION (Exam Ready)

  • Call by value ‚Üí copy passed
  • Call by reference ‚Üí address passed
  • Recursion ‚Üí function calls itself
  • Base condition ‚Üí recursion stop
  • Stack ‚Üí function call storage

Structure ek user-defined data type hoti hai jiska use different data types ko ek single unit me store karne ke liye kiya jata hai.

Simple words:

Structure = multiple variables ko ek naam ke andar rakhna

Example:

Student ka record:
name (string), roll no (int), marks (float)

6.1 Structure

Structure Definition

struct Student {
    int roll;
    char name[20];
    float marks;
};

Yahan struct Student ek new data type ban gaya.

Structure Variable Declaration

struct Student s1;

Ya declaration ke sath:

struct Student {
    int roll;
    char name[20];
    float marks;
} s1, s2;

Access Structure Members

Structure ke members ko access karne ke liye dot (.) operator ka use hota hai. 

s1.roll = 101;
s1.marks = 85.5;

User-defined Data Type

Structure ko user-defined data type isliye kaha jata hai kyunki programmer khud apna data type create karta hai.

Example:

struct Book {
    int pages;
    float price;
};

Ab struct Book ek new data type hai.

Structure Array

Jab multiple structure variables chahiye hote hain, tab structure ka array use karte hain.

Example:

struct Student s[3];

Access:

s[0].roll = 1;
s[1].marks = 90.5;

Use case:

Multiple students ka data store karna.

Structure Pointer

Structure pointer ka use structure ke address ko store karne ke liye hota hai.

Example:

struct Student s1;
struct Student *ptr;

ptr = &s1;

Access members using pointer:

ptr->roll = 10;
ptr->marks = 88.5;

Note:

-> operator structure pointer ke sath use hota hai.

Nested Structure

Jab ek structure ke andar doosra structure hota hai, use nested structure kehte hain.

Example:

struct Address {
    char city[20];
    int pincode;
};

struct Student {
    int roll;
    struct Address add;
};

Access:

s1.add.city;
s1.add.pincode;

6.2 Union

Union bhi structure jaisa hota hai, lekin sab members same memory location share karte hain.

Difference:

  • Structure ‚Üí sab members ke liye alag memory
  • Union ‚Üí sab members ke liye same memory

Example:

union Data {
    int i;
    float f;
};

Note:

Union me ek time par sirf ek member hi value hold karta hai.

6.3 Enum (Enumeration)

Enum ek user-defined data type hota hai jisme fixed named constants hote hain.

Example:

enum Day {Mon, Tue, Wed, Thu, Fri};

Use:

enum Day d;
d = Mon;

Default values:

Mon = 0, Tue = 1, Wed = 2 ...

6.4 Structure vs Union vs Enum

  • Structure ‚Üí different data types, more memory
  • Union ‚Üí memory efficient
  • Enum ‚Üí named constant values

6.5 Common Mistakes

  • Union ko structure jaisa samajhna ‚ùå
  • Structure pointer me dot (.) use karna ‚ùå
  • Enum ko string value dena ‚ùå

6.6 One-Line Revision (Exam Ready)

  • Structure = collection of variables
  • Union = shared memory data type
  • Enum = named constant values

7. File Handling (Real-world Usage)

What is File Handling?

File handling ka use program ke data ko permanently store karne ke liye hota hai.

Simple words:

File Handling = data ko file me read aur write karna

Real-world examples:

  • Student record save karna
  • Employee salary report
  • Login system me user data

7.1 File Basics

C language me file handling <stdio.h> header file ke through hoti hai.

File ke saath kaam karne ke liye FILE pointer use hota hai.

File Pointer

FILE *fp;

Explanation:

  • FILE ‚Üí predefined structure hai
  • fp ‚Üí file ka address store karta hai

7.2 File Modes

File mode batata hai file kis purpose ke liye open ho rahi hai.

  • r ‚Üí read (file exist honi chahiye)
  • w ‚Üí write (new file create / old delete)
  • a ‚Üí append (data end me add)
  • r+ ‚Üí read + write
  • w+ ‚Üí write + read

Example:

fp = fopen("data.txt", "r");

7.3 File Functions

fopen()

File ko open karne ke liye use hota hai.

fp = fopen("data.txt", "w");

Note:

Agar file open nahi hoti to fopen() NULL return karta hai.

fclose()

Open file ko close karne ke liye use hota hai.

fclose(fp);

File close karna important hota hai memory free karne ke liye.

fprintf()

File me formatted data write karne ke liye use hota hai.

fprintf(fp, "Marks = %d", marks);

Kaam karta hai jaise printf(), bas output file me jata hai.

fscanf()

File se formatted data read karne ke liye use hota hai.

fscanf(fp, "%d", &marks);

Kaam karta hai jaise scanf(), bas input file se aata hai.

fgets()

File se ek line read karne ke liye use hota hai.

fgets(str, 50, fp);

Safe function hai, buffer overflow ka risk kam hota hai.

fputs()

File me string write karne ke liye use hota hai.

fputs("Hello C File", fp);

fread() & fwrite()

Binary files ke liye use hote hain.

fwrite(&obj, sizeof(obj), 1, fp);
fread(&obj, sizeof(obj), 1, fp);

Use case:

  • Image file
  • Database file
  • Binary data storage

7.4 File Handling Flow

  • File open ‚Üí fopen()
  • Read / Write ‚Üí file functions
  • File close ‚Üí fclose()

7.5 Common Mistakes

  • File open check na karna ‚ùå
  • fclose() bhool jana ‚ùå
  • Wrong file mode use karna ‚ùå
  • Binary file me fprintf use karna ‚ùå

7.6 One-Line Revision (Exam Ready)

  • File pointer = FILE *fp
  • fopen() ‚Üí open file
  • fclose() ‚Üí close file
  • fprintf / fscanf ‚Üí formatted I/O
  • fread / fwrite ‚Üí binary data

8. Preprocessor & Macros (C Programming)

What is Preprocessor?

Preprocessor C program ka wo part hota hai jo compilation se pehle execute hota hai.

Simple words:

Preprocessor = compiler se pehle kaam karne wala program

Preprocessor ka kaam:

  • Header files include karna
  • Macros define karna
  • Conditional compilation control karna

8.1 Preprocessor Directives

Preprocessor directives hamesha # symbol se start hote hain.

#define

#define ka use constant ya macro define karne ke liye hota hai.

Example (Constant):

#define PI 3.14

Use:

area = PI * r * r;

Example (Macro):

#define SQUARE(x) x*x

Use:

SQUARE(5)  ‚Üí 25

#include

#include ka use header files add karne ke liye hota hai.

Example:

#include <stdio.h>
#include "myfile.h"

Difference:

  • < > ‚Üí system header files
  • " " ‚Üí user-defined header files

#undef

#undef ka use defined macro ko remove karne ke liye hota hai.

Example:

#define MAX 100
#undef MAX

Ab MAX ka use nahi kar sakte.

8.2 Macros vs Functions

Macro aur function ka kaam similar lagta hai, lekin internally dono different hote hain.

  • Macro ‚Üí preprocessor ke time expand hota hai
  • Function ‚Üí runtime par call hota hai

Comparison:

  • Macro ‚Üí faster execution
  • Function ‚Üí memory efficient
  • Macro ‚Üí no type checking
  • Function ‚Üí type checking supported

Example:

#define ADD(a,b) a+b

Vs

int add(int a, int b) {
    return a + b;
}

8.3 Conditional Compilation

Conditional compilation ka use specific condition ke basis par code compile karne ke liye hota hai.

#ifdef

Check karta hai ki macro defined hai ya nahi.

Example:

#define DEBUG

#ifdef DEBUG
printf("Debug mode ON");
#endif

#ifndef

Check karta hai ki macro defined nahi hai.

Example:

#ifndef LIMIT
#define LIMIT 10
#endif

Use case:

Header files ko multiple inclusion se bachane ke liye.

8.4 Common Mistakes

  • Macro me brackets na lagana ‚ùå
  • Macro ko function samajhna ‚ùå
  • #include syntax galat likhna ‚ùå

8.5 One-Line Revision (Exam Ready)

  • Preprocessor ‚Üí compiler se pehle kaam karta hai
  • #define ‚Üí macro define
  • #include ‚Üí header file add
  • #ifdef / #ifndef ‚Üí conditional compilation

9. Advanced C Concepts

Overview

Advanced C concepts ka use program ko efficient, flexible aur powerful banane ke liye hota hai.

Is chapter me hum seekhenge:

  • Memory behavior
  • Program execution control
  • Low-level operations
  • Error handling

9.1 Storage Classes

Storage class batata hai:

  • Variable ki scope
  • Lifetime
  • Visibility
  • Memory location

auto

auto default storage class hoti hai.

Example:

auto int x = 10;

Note:

auto keyword usually likhne ki zarurat nahi hoti.

static

static variable apni value program ke end tak retain karta hai.

Example:

void fun() {
    static int count = 0;
    count++;
    printf("%d", count);
}

Use case:

Function calls ke beech value preserve karna.

extern

extern ka use global variable ko dusri file me access karne ke liye hota hai.

Example:

extern int total;

Note:

Memory allocation nahi hota, sirf reference hota hai.

register

register variable CPU register me store karne ka request karta hai.

Example:

register int i;

Note:

Register variable ka address (&) nahi le sakte.

9.2 Command Line Arguments

Command line arguments ka use program ko runtime par input dene ke liye hota hai.

Main function:

int main(int argc, char *argv[])

  • argc ‚Üí arguments count
  • argv ‚Üí arguments values

Example:

./program 10 20

Note:

argv[0] hamesha program ka naam hota hai.

9.3 typedef

typedef ka use existing data type ko naya naam dene ke liye hota hai.

Example:

typedef unsigned int uint;
uint a = 10;

Use case:

Code readability improve karna.

9.4 Bit Manipulation

Bit manipulation ka use low-level programming aur performance optimization ke liye hota hai.

Bitwise Operators

  • & ‚Üí AND
  • | ‚Üí OR
  • ^ ‚Üí XOR
  • << ‚Üí Left shift
  • >> ‚Üí Right shift
  • ~ ‚Üí NOT

Example:

int a = 5;   // 0101
int b = 3;   // 0011
a & b = 1;   // 0001

Flags

Flags ka use multiple ON/OFF states ko ek single variable me manage karne ke liye hota hai.

Example:

#define READ  1
#define WRITE 2
#define EXEC  4

Use:

int permission = READ | WRITE;

9.5 Error Handling

C me built-in exception handling nahi hoti, isliye error handling manually hoti hai.

errno

errno ek global variable hai jo error code store karta hai.

Example:

if (fp == NULL) {
    printf("%d", errno);
}

perror()

perror() readable error message print karta hai.

Example:

perror("File error");

Output system-defined message deta hai.

9.6 Common Mistakes

  • static aur global variable confuse karna ‚ùå
  • argv[0] ko input samajhna ‚ùå
  • Bitwise aur logical operator confuse karna ‚ùå
  • errno ko manually change karna ‚ùå

9.7 One-Line Revision (Exam Ready)

  • Storage class ‚Üí scope & lifetime
  • argc/argv ‚Üí runtime input
  • typedef ‚Üí alias for data type
  • Bit manipulation ‚Üí low-level control
  • errno/perror ‚Üí error handling

10. Data Structures in C (Career Turning Point)

What is Data Structure?

Data structure ka use data ko efficient way me store, manage aur access karne ke liye hota hai.

Simple words:

Data Structure = data ko sahi tareeke se organize karna

Career point:

  • DSA interviews ka base
  • Performance optimization
  • Problem solving skills

10.1 Linked List

Linked list ek linear data structure hoti hai jisme elements nodes ke form me hote hain.

Har node me hota hai:

  • Data
  • Next node ka address

Properties:

  • Dynamic size
  • Non-contiguous memory
  • Insertion easy

Types:

  • Singly Linked List
  • Doubly Linked List
  • Circular Linked List

10.2 Stack

Stack ek linear data structure hoti hai jo LIFO principle follow karti hai.

LIFO:

Last In First Out

Operations:

  • Push ‚Üí element add
  • Pop ‚Üí element remove
  • Peek ‚Üí top element

Real-world example:

  • Plate stack
  • Undo/Redo

10.3 Queue

Queue ek linear data structure hoti hai jo FIFO principle follow karti hai.

FIFO:

First In First Out

Operations:

  • Enqueue ‚Üí insert
  • Dequeue ‚Üí delete

Real-world example:

  • Bank line
  • Ticket counter

10.4 Circular Queue

Circular queue normal queue ka advanced version hota hai.

Difference:

  • Last position first se connected hoti hai
  • Memory wastage kam hota hai

Use case:

  • CPU scheduling
  • Buffer management

10.5 Trees (Basic)

Tree ek non-linear data structure hota hai.

Basic terms:

  • Root
  • Parent
  • Child
  • Leaf

Binary Tree:

  • Max 2 children

Use case:

  • File system
  • Hierarchy data

10.6 Searching

Searching ka use data structure me element find karne ke liye hota hai.

Types:

  • Linear Search
  • Binary Search

Difference:

  • Linear ‚Üí slow, simple
  • Binary ‚Üí fast, sorted data required

10.7 Sorting

Sorting ka use data ko order me arrange karne ke liye hota hai.

Types:

  • Bubble Sort
  • Selection Sort
  • Insertion Sort

Basic idea:

  • Bubble ‚Üí adjacent swap
  • Selection ‚Üí minimum select
  • Insertion ‚Üí correct position insert

10.8 Comparison Table (Exam Ready)

  • Stack ‚Üí LIFO
  • Queue ‚Üí FIFO
  • Circular Queue ‚Üí memory efficient
  • Tree ‚Üí hierarchical structure

10.9 Common Mistakes

  • Stack aur queue confuse karna ‚ùå
  • Binary search unsorted array me lagana ‚ùå
  • Linked list ko array samajhna ‚ùå

10.10 One-Line Revision (Exam Ready)

  • Linked list = dynamic structure
  • Stack = LIFO
  • Queue = FIFO
  • Tree = non-linear
  • Sorting = arrange data
