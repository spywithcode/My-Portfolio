## ðŸš€ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- Internet connection for external libraries

### Installation & Usage

1. **Clone the repository**:
   ```bash
   git clone https://github.com/spywithcode/My-Portfolio.git
   cd My-Portfolio
   ```

2. **Open in browser**:
   - Simply open `index.html` in your web browser
   - Or use a local server for better experience:
     ```bash
     # Using Python
     python -m http.server 8000

     # Using Node.js
     npx serve .

     # Then visit http://localhost:8000
     ```

3. **Navigate the site**:
   - Use the sidebar navigation to switch between sections
   - Try the theme switcher in the bottom right corner
   - Click on portfolio projects to view GitHub repos and live demos

## ðŸŽ¨ Customization

### Changing Themes
- Click the gear icon in the bottom right to open theme switcher

- Edit `JS/portfolio.js` to add new projects to the arrays

- Ensure GitHub and live demo links are valid

### Modifying Content
- Update personal information in `JS/about.js`
- Modify services in `JS/services.js`
- Change home content in `JS/home.js`
